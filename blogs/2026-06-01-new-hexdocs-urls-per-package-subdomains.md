HexDocs URLs are changing for security. Public package docs move from hexdocs.pm/package to package.hexdocs.pm, and private organization docs move from org.hexdocs.pm/package to org.hexorgs.pm/package (a new top-level domain). This isolates packages from each other in the browser, addressing a finding from the recent security audit.
